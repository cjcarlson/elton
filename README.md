Installing the package
----------------------

Install directly from Github:

``` r
knitr::opts_chunk$set(echo = FALSE)

# If you don't have devtools:
# install.packages("devtools")

devtools::install_github("cjcarlson/elton")
```

``` r
# Load the package

library(elton)
```
